<h1 align="center">TS Fullstack Template</h1>

This is a template project for typescript full-stack project.

# Technologies
- TypeScript
- React
- Express
- Mongo DB
- Prisma (ORM)
- jest (testing lib)
- eslint
- Apollo server
- Apollo client
- GraphQL Code Generator (auto generate TypeScript types)
- vite (build tool)
- tailwind (css)
